Global Feed Widget

Embed the global news feed on any website. This widget shows the latest articles from all sources.

Embed Code:

<iframe src="http://localhost:8080/widget/global" width="100%" height="400" frameborder="0"> </iframe>

Compact Global Feed

A more compact version perfect for sidebars.

<iframe src="http://localhost:8080/widget/global?compact=true&limit=3" width="100%" height="300" frameborder="0"> </iframe>

Dark Theme Widget

A dark-themed version for dark websites.

<iframe src="http://localhost:8080/widget/global?theme=dark&compact=true&limit=3" width="100%" height="300" frameborder="0"> </iframe>

Personal Feed Widget

Show a personalized feed for a specific Bluesky user. Replace 'bsky.app' with any valid handle.

Embed Code:

<iframe src="http://localhost:8080/widget/personal?user=YOUR_HANDLE_HERE" width="100%" height="400" frameborder="0"> </iframe>

Customization Options

You can customize the widgets using URL parameters:

  • theme=dark - Use dark theme
  • limit=10 - Number of articles (1-100, default: 20)
  • compact=true - Use compact layout
  • autorefresh=300 - Auto-refresh interval in seconds (min: 60)
  • user=handle.bsky.social - Specific user for personal feeds

Example with all options:

<iframe src="http://localhost:8080/widget/global?theme=dark&limit=5&compact=true&autorefresh=300" width="100%" height="400" frameborder="0"> </iframe>

Usage Instructions

To embed these widgets on your website:

  1. Copy the embed code from any example above
  2. Replace localhost:8080 with your open.news server URL
  3. Adjust the width and height attributes as needed
  4. Customize using the URL parameters listed above
  5. Paste the code into your HTML where you want the feed to appear

For WordPress, Squarespace, or other CMS: Use the HTML/Custom Code block to insert the iframe code.
